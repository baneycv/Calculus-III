  1. 1.

    Let f⁢(x,y)=sin⁡(1x⁢y). Find the domain and range of f.
    Solution: Starting with the inputs of f, we note that x≠0 and y≠0 otherwise the input to the function will be undefined. The range or output of the sine function is always between −1 and 1. Therefore, −1≤f⁢(x,y)≤1.

  2. 2.

    Let f⁢(x,y)=xx−y. For which values of x is f defined?
    Solution: The function is defined when the expression under the square root is positive, i.e.,

    x−y>0⇒y<x.

    So for each fixed x, the function is defined for all y<x.

  3. 3.

    Let f⁢(x,y)=y2−x2y2+x2. Determine lim(x,y)→(0,0)f⁢(x,y) when y=3⁢x and y=x2. Is f continuous at (0,0)? Explain.
    Solution: We first note that (0,0) is not in the domain of the function because it will make the denominator equal to zero. While it does make the numerator equal to zero also, this only indicates that you would have an indeterminate form. This would mean then that the limit could potentially exist, but the point may not be in the range of f.

    To compute the limits, we start by letting y=3⁢x and plugging this in to the function. Thus, we have

    lim(x,y)→(0,0)f⁢(x,y) = limx→0f⁢(x,3⁢x)
    = limx→0(3⁢x)2−x2(3⁢x)2+x2
    = limx→08⁢x210⁢x2
    = 45

    To compute the second limit, we let y=x2.

    lim(x,y)→(0,0)f⁢(x,y) = limx→0f⁢(x,x2)
    = limx→0(x2)2−x2(x2)2+x2
    = limx→0x4+x2x4−x2
    = limx→0x2⁢(x2+1)x2⁢(x2−1)
    = limx→0x2+1x2−1
    = −1

    Thus, since (0,0) is not in the domain of the function, and the limit approaches different values as we travel different paths, the function f is not continuous at (0,0).

  4. 4.

    Let f⁢(x,y)=x−4⁢y6⁢y+7⁢x. Determine lim(x,y)→(0,0)f⁢(x,y) when y=0 and x=0. Does lim(x,y)→(0,0)f⁢(x,y) exist? Explain. Is f continuous at (0,0)? Explain.
    Solution: Along y=0:

    lim(x,y)→(0,0)f⁢(x,y) = limx→0f⁢(x,0)
    = limx→0x7⁢x
    = 17

    Along x=0:

    lim(x,y)→(0,0)f⁢(x,y) = limx→0f⁢(0,y)
    = limy→0−4⁢y6⁢y
    = −23

    Since the limits along different paths yield different results, the limit does not exist. Also, since both the limit does not exist at (0,0), and that (0,0) is not in the domain of f, we can conclude that f is not continuous at (0,0).

    Consider the function w=f⁢(y−x+a,y−z+b,z−x+c) where f is an arbitrary function of three variables, and a, b, and c are constants. Assume the partial derivatives of f exists.

    1. (a)

      Compute the following sum:

      A=∂w∂x+∂w∂y+∂w∂z

      Solution: Let u=y−x+a, v=y−z+b, and s=z−x+c. Then w=f⁢(u,v,w). Thus, by the Chain Rule we have the following

      ∂w∂x = ∂w∂u⁢∂u∂x+∂w∂v⁢∂v∂x+∂w∂s⁢∂s∂x
      = ∂w∂u⋅−1+∂w∂v⋅0+∂w∂s⋅−1
      = −∂w∂u−∂w∂s

      Next,

      ∂w∂y = ∂w∂u⁢∂u∂y+∂w∂v⁢∂v∂y+∂w∂s⁢∂s∂y
      = ∂w∂u⋅1+∂w∂v⋅1+∂w∂s⋅0
      = ∂w∂u+∂w∂v

      Lastly,

      ∂w∂z = ∂w∂u⁢∂u∂z+∂w∂v⁢∂v∂z+∂w∂s⁢∂s∂z
      = ∂w∂u⋅0+∂w∂v⋅−1+∂w∂s⋅1
      = −∂w∂v+∂w∂s

      Thus, we have that

      A = ∂w∂x+∂w∂y+∂w∂z
      = (−∂w∂u−∂w∂s)+(∂w∂u+∂w∂v)+(−∂w∂v+∂w∂s)
      = 0
    2. (b)

      If x=sin⁡t, y=cos⁡t, and z=et, determine d⁢wd⁢t.
      Solution: Using the appropriate Chain Rule,

      d⁢wd⁢t = ∂w∂x⁢d⁢xd⁢t+∂w∂y⁢d⁢yd⁢t+∂w∂z⁢d⁢zd⁢t
      = ∂w∂x⁢(cos⁡t)+∂w∂y⁢(−sin⁡t)+∂w∂z⁢(et)

    1. (a)

      Consider the function f⁢(x,y)=sin⁡(x⁢y). Find the direction in which f increases most rapidly at the point (x0,y0)=(1,π). What is the maximal directional derivative at the point?
      Solution: The direction of greatest increase of f at a point (x0,y0) is in the direction of the gradient vector, ∇f⁢(x0,y0). Hence, we have

      ∇f⁢(x,y)=(y⁢cos⁡(x⁢y),x⁢cos⁡(x⁢y)).

      Thus, we have that,

      ∇f⁢(1,π) = (π⁢cos⁡(π),1⁢cos⁡(π))
      = (−π,−1)

      The maximal directional derivative is the Euclidean norm of the gradient vector. Thus, we have

      ∥∇f⁢(1,π)∥=π2+1

      Remark. Remember a directional derivative is measured by determining the projection of the gradient vector onto a unit vector in the direction we are heading. That is, the directional derivative of f in the direction of 𝐮 is given by D𝐮⁢f⁢(x,y)=∇f⋅𝐮, where 𝐮 is a unit vector. Thus, the directional derivative is a scalar value. The gradient is a vector.

    2. (b)

      Now consider the function f⁢(x,y,z)=sin⁡(x⁢y)+cos⁡(x⁢z). Find the equation of the plane tangent to the level surface f⁢(x,y,z)=0 at the point (x0,y0,z0)=(π/4,4,2).
      Solution: The equation of a plane is determined by the normal vector 𝐧 and a point in the plane. We have that for (x,y,z) in the plane, we have then that,

      𝐧⋅(x−x0,y−y0,z−z0)=0

      We know that the vector normal to the surface at the point is the gradient vector of the surface to that point. Thus, we first need to find the gradient, evaluate it at the point, and set up our equation for the plane. Hence, we have that

      ∇f⁢(x,y,z)=(y⁢cos⁡(x⁢y)−z⁢sin⁡(x⁢y),x⁢cos⁡(x⁢y),−x⁢sin⁡(x⁢z))

      Thus,

      ∇f⁢(π/4,4,2) = (4⁢cos⁡(π)−2⁢sin⁡(π/2),π/4⋅cos⁡(π),−π/4⋅sin⁡(π/2))
      = (−6,−π4,−π4)

      Thus, the equation of the plane tangent to the surface f would be given by

      (−6,−π4,−π4)⋅(x−π4,y−4,z−2)=0

      or

      −6⁢(x−π4)−π4⁢(y−4)−π4⁢(z−2)=0

    2. (b)

      Use the tangent plane approximation at the point (1,2) to approximate the value of f⁢(0.95,2.1).
      Solution:. We need to find the tangent plane to f at the point (1,2), and then solve for z, and evaluate at the point (0.95,2.1). To find the tangent plane, we treat f as a level surface. It is going to look like alphabet soup the way this is done. The idea behind it is that we need to create a function w⁢(x,y,z). We do this by letting f⁢(x,y)=z

      f⁢(x,y) = x2⁢y+x⁢y+y3
      z = x2⁢y+x⁢y+y3
      w = x2⁢y+x⁢y+y3−z

      Before we proceed, we will also need to know z at the point (1,2). Thus, f⁢(1,2)=12. Proceeding with finding the plane tangent to the surface, we find the gradient of w.

      ∇w=(2⁢x⁢y+y,x2+x+3⁢y2,−1)

      Evaluating the gradient at (1,2,12), we have,

      ∇w⁢(1,2,12)=(6,14,−1)

      This is the normal to the plane (and the surface) at this point. The equation of the plane is the set of all vectors whose dot product with the normal is equal to 0. Thus, the equation of the plane is given by

      ∇w⁢(1,2,12)⋅(x−1,y−2,z−12) = 0
      (6,14,−1)⋅(x−1,y−2,z−12) = 0
      6⁢(x−1)+14⁢(y−2)−(z−12) = 0

      Solving for z, we have that z=6⁢(x−1)+14⁢(y−2)+12. Using the equation of the tangent plane, we evaluate at the desired point of approximation,

      z⁢(0.95,2.1) = 6⁢(0.95−1)+14⁢(2.1−2)+12
      = −.03+1.4+12
      = 13.37

      The actual value is f⁢(0.95,2.1)≈13.151. Thus our approximation is fairly close.
      Remark. It is worth remembering that the first order approximation (or first degree Taylor polynomial in multiple dimensions) is given by

      f⁢(x0+h,y0+k)≈f⁢(x0,y0)+fx⁢(x0,y0)⁢h+fy⁢(x0,y0)⁢k

      This formula is very frequently applied in Engineering and Physics appllications.
